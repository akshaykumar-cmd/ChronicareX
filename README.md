# ChronicareX

ChronicareX is an AI-powered chronic disease care planner designed to support
patients with diabetes and hypertension using safe, guideline-based insights.

## Problem Statement
Patients with chronic diseases often find it difficult to consistently follow
daily care plans, track health metrics, and understand when medical attention
is required.

## Goal
To help patients follow personalized daily care plans, track progress,
and adapt recommendations safely using AI.

## Proposed Solution
ChronicareX provides:
- Personalized daily care tasks (glucose/BP checks, activity, diet)
- Trend detection from health logs
- Weekly care plan adjustments
- Motivational and multilingual messages
- Safety-first alerts when values exceed thresholds

## Key Features
- Guideline-based thresholds (ADA / AHA)
- Synthetic data for privacy protection
- Structured outputs for easy integration
- Clear escalation prompts for critical values

## Prototype Description
This repository contains a working prototype that demonstrates how AI logic
can analyze patient logs, detect trends, and update care plans weekly.

## Safety Disclaimer
⚠️ This system is NOT a medical diagnosis or treatment tool.
It is an educational prototype only. Always consult a healthcare professional.

## Data & Privacy
- Uses synthetic / mock data only
- No real patient information is stored or processed
